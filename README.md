# go.davsk.net
This is a list of the **Go** packages from Davsk Ltd Co that are available to import.

## Usage

### Prerequisites
``` zsh
go get -u go.bobheadxi.dev/twist
```

### Edit the List
``` zsh
kate twist.yml
```

### Update the list
``` zsh
twist -c twist.yml
```

### Verify the updates
If the changes are correct then commit and push.
